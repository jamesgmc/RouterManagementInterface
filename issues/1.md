## Title
Add sorting functionality to Packets Received and Packets Sent columns

## Description
The Packets Received and Packets Sent columns in the router hosts table should be sortable, similar to the other columns (Type, Blacklisted, IP, MAC, DHCP). Currently, these columns do not have click handlers to trigger sorting.

### File:
RouterManagement/Pages/Index.razor

### Acceptance Criteria:
- Users can click on the Packets Received column header to sort by packets received
- Users can click on the Packets Sent column header to sort by packets sent
- Sorting should toggle between ascending and descending
- Visual indicator (↑/↓) should display the current sort direction
- Sorting should work consistently with existing column sort functionality
